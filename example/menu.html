Basic Menu

All versions of the menu are a <li> filled with <li> elements containing links. By default, a Menu is horizontally oriented.

<ul class="menu">
  <li><a href="#">Item 1</a></li>
  <li><a href="#">Item 2</a></li>
  <li><a href="#">Item 3</a></li>
  <li><a href="#">Item 4</a></li>
</ul>
